# Call for Founding Members

**The OOXML Behavioral Intelligence Project is seeking founding technical partners to define the first open behavioral specification of OOXML.**

This is a vendor-neutral, open-source initiative. We are not building another office suite. We are building **the knowledge layer that makes every office suite better.**

---

## What Does It Mean to Be a Founding Member?

Founding members contribute to the project in its formative stage and help shape:

1. **The compatibility case format** — the universal test case schema for OOXML behavior
2. **The knowledge base** — the first structured corpus of behavioral data
3. **The testing pipeline** — automated cross-implementation comparison
4. **The governance model** — vendor-neutral, community-driven

Founding members are recognized in:
- The `CITATION.cff` file
- The repository README
- Academic publications
- Project documentation
